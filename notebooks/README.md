# Jupyter Notebook Tutorials

Interactive tutorials for learning threat hunting with AI.

1. **01_getting_started.ipynb** - Basic scenario generation and analysis
2. **02_detection_rules.ipynb** - Writing and testing detection rules (coming soon)

## Setup

```bash
pip install jupyter notebook
jupyter notebook
```

Then open any notebook and run the cells!
